# LLAMA E 4.0.6（ローカル実装）

公開版 [LLAMA E 4.0.6](https://www.llamatests.org/tests/LLAMA_E.html) の課題フローを、ローカル実験用に再実装したものです。

## 対応内容

- UI 言語: 日本語デフォルト、日本語 / 英語切替
- 参加者情報: 氏名と ID を入力。ID は 1〜80 文字で、日本語・小文字・数字 1 桁も可
- 音量確認画面あり
- 学習時間: 2 分
- `テスト開始` 有効化: 開始 60 秒後
- 学習項目: 公式版と同じ 24 音価ボタン
- テスト項目: 公式版と同じ 20 音声と 21 選択肢
- 採点: 20 点満点。正答列は公式 CGI で満点確認済み
- 出力: `.xlsx`、SheetJS が使えない場合は CSV

## 使い方

```bash
cd LLAMA_E
python3 -m http.server 8766
```

ブラウザで `http://127.0.0.1:8766/` を開きます。

## 結果ファイル

`Meta`、`Wide`、`Trials`、`Events` の 4 シートを出力します。`Trials` には項目ごとの開始時刻、回答保存時刻、選択 ID、正答 ID、得点、RT、音声 URL が入ります。
